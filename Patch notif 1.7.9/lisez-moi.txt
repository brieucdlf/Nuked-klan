 • Ce patch contient • 

Un fichier js : user.js
Un fichier css : user.css
Une image : notif-images.png
Un fichier lang : french_lang.php




 • Ce patch contient deux fichiers Users •

 - Le premier valide avec le patch nk-convertion 1.7.9.

 - Le deuxième valide sans patch.



 • FONCTIONNALITES •

  - Affichage de l'adresse ip lors du login.
  
  - Affichage du nombre de message(s) privé(s) lors du login.

  - Affichage en css lors du login ou du message d'erreur lors d'une saisie incorrecte.




 • INSTALLATION • 

 - Uploadez l'un des deux fichiers User dans modules/User/ de votre ftp
